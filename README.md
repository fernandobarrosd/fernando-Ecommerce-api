# Fernando ECommerce API

Uma API de ecommerce, com a possibilidade de poder criar produtos, criar pedidos e efetuar autenticação via JWT.


## Funcionalidades
- Criar pedidos
- Criar produtos
- Criar admins
- Criar clientes
- Autenticação via JWT
- Login com e-mail e senha
